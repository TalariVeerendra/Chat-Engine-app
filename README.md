# chat_app

This project is a fully functional Chat Application built with Flutter, powered by Firebase for backend services. Designed for real-time communication, the app provides a smooth and secure chatting experience with live message updates, user authentication, and powerful account management features.

Using Firebase Authentication, Cloud Firestore, and Firebase Storage, the app supports account creation, updates, deletion, and real-time synchronization of chat data. Users can update their profile information, change profile pictures, and see changes reflected instantly across all connected devices.
